# Security Policy

## Supported Versions

We will provide security fixes for the latest version of Orbix and encourage Orbix users through auto-updates to use the latest version of the app.

## Reporting a Vulnerability

Please file security vulnerabilities by using [report a vulnerability](https://github.com/K4ran909/Orbix/security/advisories/new). Please do not file security vulnerabilities as a regular issue as the information could be used to exploit Orbix users.
